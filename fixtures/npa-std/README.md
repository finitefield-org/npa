# npa-std Fixture

This is the SRA-02 local package fixture for the first standalone `npa-std`
activation step.

The fixture intentionally contains only the legacy compatibility modules needed
by the current proof corpus and `npa-mathlib` Layer 0:

```text
Std.Logic.Eq
Std.Nat.Basic
```

The checked-in `source.npa` files are source-package skeletons. They fix module
membership and import intent for package tooling, but certificate contents are
generated by the deterministic Rust core-module builder selected by:

```text
producer_profile = "std-library-legacy-core-builder"
```

Trusted proof evidence remains the canonical `.npcert` bytes and source-free
checker verdicts. The source skeletons, package manifest, theorem index,
publish plan, and CI status are package metadata, not proof evidence.
